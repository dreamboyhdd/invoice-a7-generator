# Invoice A7 Generator

Ứng dụng tạo hóa đơn in khổ A7 (74mm) với chiều dài tự động theo nội dung.

## Tính năng

- ✅ Tạo hóa đơn khổ A7 (74mm width, auto height)
- ✅ Thêm/xóa sản phẩm
- ✅ Tính toán tự động tổng tiền, chiết khấu
- ✅ Tùy chỉnh logo, thông tin cửa hàng
- ✅ Lưu dữ liệu tự động (localStorage)
- ✅ In hoặc xuất PDF
- ✅ Giao diện mobile-friendly
- ✅ Dark/Light mode

## Demo

[Live Demo](https://yourusername.github.io/invoice-a7-generator/)

## Cách sử dụng

1. Mở file `invoice_a7_generator.html` trên trình duyệt
2. Nhập thông tin sản phẩm và thêm vào danh sách
3. Xem preview hóa đơn
4. Bấm "In hóa đơn" và chọn "Lưu dưới dạng PDF"

## Cài đặt

Chỉ cần tải file `invoice_a7_generator.html` và mở bằng trình duyệt. Không cần cài đặt gì thêm.

## Công nghệ

- HTML5
- Tailwind CSS + DaisyUI
- Vanilla JavaScript
- LocalStorage API
